# DotaWiki Ultra

Цветной статический сайт-wiki о Dota 2.

## Что внутри

- Главная страница с красивым hero-блоком.
- Большая база героев с изображениями, фильтрами и избранным.
- Карточки предметов с картинками, ценами, категориями и поиском.
- Страницы механик, гайдов, сборок, словаря, патчей и источников.
- Тёмная/светлая тема.
- Адаптив под телефон.

## Как опубликовать на GitHub Pages

1. Распакуй архив.
2. Открой свой репозиторий `dota-wiki`.
3. Нажми **Add file → Upload files**.
4. Перетащи все файлы и папки из распакованной папки.
5. Нажми **Commit changes**.
6. GitHub Pages обновит сайт через 1–3 минуты.

## Важно

Это учебный фан-проект. Dota 2 принадлежит Valve. Картинки героев и предметов подгружаются с публичного Steam CDN. Баланс, цены и патчноуты могут меняться, поэтому перед серьёзной публикацией сверяй данные с официальными источниками.
