# DotaWiki

Статический сайт об основных разделах Dota 2: герои, предметы, механики, роли, сборки, словарь и патчи.

## Файлы

- `index.html` — главная страница.
- `heroes.html` — список героев.
- `hero.html` — страница отдельного героя.
- `items.html` — предметы.
- `mechanics.html` — механики.
- `roles.html` — роли и стадии матча.
- `builds.html` — сборки по ролям.
- `glossary.html` — словарь.
- `patches.html` — последние патчи.
- `sources.html` — источники и правовая информация.
- `404.html` — страница ошибки.
- `js/data.js` — база данных сайта.
- `js/app.js` — логика поиска, фильтров, избранного и темы.
- `css/style.css` — оформление.

## Публикация на GitHub Pages

Загрузите содержимое папки в корень репозитория и включите Pages: `Settings → Pages → Deploy from a branch → main → /root`.

Текущая версия: 2.4.
