# DotaWiki

Статический сайт о Dota 2: герои, предметы, механики, роли, сборки, сравнение, словарь и патчи.

## Разделы

- `index.html` — главная страница.
- `heroes.html` — список героев.
- `hero.html` — страница героя.
- `items.html` — список предметов.
- `item.html` — страница предмета.
- `mechanics.html` — механики.
- `roles.html` — роли и стадии матча.
- `builds.html` — сборки по ролям.
- `compare.html` — сравнение героев.
- `glossary.html` — словарь.
- `patches.html` — патчи.
- `sources.html` — источники и правовая информация.
- `404.html` — страница ошибки.

## Публикация

Загрузите содержимое папки в корень репозитория и включите GitHub Pages: `Settings → Pages → Deploy from a branch → main → /root`.

Текущая версия: 2.6.
