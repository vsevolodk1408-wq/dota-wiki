# DotaWiki

Статический сайт-энциклопедия по Dota 2.

## Разделы

- Главная страница
- Герои
- Предметы
- Механики
- Роли
- Сборки
- Словарь
- Патчи
- Источники

## Публикация на GitHub Pages

1. Загрузите содержимое папки проекта в корень репозитория.
2. Откройте Settings → Pages.
3. Выберите Source: Deploy from a branch.
4. Branch: main, Folder: /root.
5. Нажмите Save.

## Данные

Карточки героев, предметов, механик, ролей и терминов находятся в `js/data.js`.
