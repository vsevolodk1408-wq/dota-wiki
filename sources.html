Источники и публикация

Проект фанатский и учебный. Для актуальной базы нужно сверять данные с официальными страницами Dota 2.

Основные источники для проверки

  • Официальная страница героев Dota 2
  • Официальные патчноуты Dota 2
  • Новости и обновления Dota 2
  • Dota 2 в Steam

Как получить публичную ссылку

  1. Создай репозиторий на GitHub.
  2. Загрузи все файлы из архива.
  3. Открой Settings → Pages.
  4. Выбери ветку main и папку /root.
  5. Через минуту GitHub выдаст ссылку вида https://username.github.io/repository/.
