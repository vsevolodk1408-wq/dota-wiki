Сайт использует внешние изображения из Steam CDN. Баланс, цены и описания стоит сверять с официальными ресурсами.

Как обновить сайт на GitHub Pages

  1. Распакуй новый архив.
  2. Открой репозиторий dota-wiki на GitHub.
  3. Нажми Add file → Upload files.
  4. Перетащи все файлы из папки сайта.
  5. Нажми Commit changes.
  6. Через пару минут GitHub Pages обновит ссылку.
